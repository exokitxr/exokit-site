What is Exokit?

Exokit is an XR engine that runs your code as native.

Power up your WebXR app

Exokit runs regular HTML+JS WebXR sites. Add hardware bindings, native code, and more with the APIs Exokit gives you access to.

Run your app in Exokit with a URL. Or, run 5 instances in multiplayer AR. Debug it all with standard DevTools.

Features

Exokit is an XR engine that runs your code as native.

Not a Browser

Exokit is an XR engine, not a web browser. It's layers of web-powered realities, not screens floating in space.

Futuristic APIs

DOM-to-canvas, 3D iframes, device emulation, scriptable bot framework. The future is already here.

Runs existing code

Don't change your code. Exokit runs regular HTML, WebXR, and WebGL, and frameworks like A-Frame and THREE.js.

Not a fork of

Firefox or Chrome

Exokit is not a fork of Chrome, Firefox, or anything else. It's fresh open source optimized for XR, by folks who live and breathe it.
